# agent-skills

A collection of agent skills for [Claude Code](https://claude.ai/code) marketplace.

## Installation

**CLI:**
```bash
claude plugin install masaki39/agent-skills
```

**Claude Code UI:**
Type `/plugin` in the chat, search for `masaki39/agent-skills`, and install.
